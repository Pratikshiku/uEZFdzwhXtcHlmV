## Graduation Employment Information Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/78f1c0d414dd43e28d00c4a4001b7efd/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Company, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Company
5. Resume
6. Announcement Information
7. Job Recruitment
8. Personal Center
9. Background Management


                             
**Company Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Resume Management
6. Announcement Information Management
7. Resume Application Management
8. Job Recruitment Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Basic Data Management (Industry Management, Job Intention Management, Announcement Information Type Management, Recruitment Position Management)
5. Company Management
6. Resume Management
7. Employment Statistics Management
8. Announcement Information Management
9. Resume Application Management
10. Student Management
11. Job Recruitment Management, Job Recruitment Message Management
12. Carousel Information
